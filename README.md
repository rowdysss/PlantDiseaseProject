Проведён анализ качества классификации, исследование ошибок моделей и тестирование устойчивости к различным условиям съёмки.

Лучшая модель DenseNet121 была интегрирована в веб-приложение на Streamlit для диагностики состояния растений по загруженному изображению листа.

Технологии: Python, TensorFlow, Keras, OpenCV, Streamlit, NumPy, Pandas.
