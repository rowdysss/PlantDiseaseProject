Проект посвящён автоматической диагностике заболеваний растений по фотографии листа с использованием методов глубокого обучения.

Выполнено обучение и сравнение пяти архитектур нейронных сетей. Лучшая модель DenseNet121 интегрирована в веб-приложение на Streamlit.

Технологии: Python, TensorFlow, Keras, OpenCV, Streamlit, NumPy, Pandas.

Автор: Александр Андреевич Дорохов.
